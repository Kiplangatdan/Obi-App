Obi App WP Ionic

This app is an example of integrating WordPress with an Ionic mobile app using the WP-API. To learn how to configure this app, please visit scottbolinger.com

Features:

  • Posts, including CPTs, taxonomies, and more
  • Pages and custom API endpoints
  • Login and hide/show membership content
  • WooCommerce REST API integration with Stripe payments
  • Tabs and/or side menu

Setup Instructions:

  1. Visit providers/configure/configure.ts and add your website url. To use WooCommerce, add your auth key as described. To use Stripe, add your publishable key in app.module.ts.
  2. Install and activate this plugin on your WordPress site: https://github.com/scottopolis/sb-app-integration
